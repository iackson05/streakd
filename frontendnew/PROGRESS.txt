PUSH NOTIFICATIONS (COMPLETE)
------------------------------
[x] Backend: services/notifications.py rewritten to support BOTH APNs + Expo tokens
    - Auto-detects token type (ExponentPushToken[...] = Expo, hex string = APNs)
    - APNs uses token-based auth (.p8 key) with JWT + HTTP/2
    - JWT is cached for 50 minutes (APNs tokens valid for 1 hour)
    - Old send_expo_push() kept as alias — all existing callers work unchanged
[x] Backend: config.py updated with APNS_KEY_ID, APNS_TEAM_ID, APNS_KEY_PATH, APNS_BUNDLE_ID, APNS_USE_SANDBOX
[x] Backend: requirements.txt updated — httpx[http2], PyJWT[crypto] added
[x] Swift: PushNotificationManager — requests permission, registers for APNs, converts
    device token to hex, uploads to backend via PUT /users/push-token
[x] Swift: AppDelegate — handles didRegisterForRemoteNotifications, foreground display, tap handling
[x] Swift: StreakdApp wires AppDelegate via @UIApplicationDelegateAdaptor
[x] Swift: OnboardingView notification step uses PushNotificationManager.requestPermission()
